## Overview

Storeroom is a user-friendly app that helps you recreate your physical storage as a neat online space: _[create storage rooms](https://storeroom-pi.vercel.app/storeroom/create)_, _add products and suppliers_, _categorize items_, and _track quantities_ in real time. Built for **homeowners**, **small businesses**, and **project teams**, Storeroom removes the guessworküòÅ. You‚Äôll always know what you have, where it is, and when you need to restock.
